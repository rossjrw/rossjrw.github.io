SCP-3211

2018

An article published on the SCP Wiki. Details a fictional entity that appears differently for each person who sees it: to emulate this for readers, I wrote many versions of the same article and present a random one to each reader. I use cookies to ensure that while every reader sees a different version, a single reader will only ever see one. The article can also only be read once. Written on a spreadsheet and compiled to JavaScript.

Ur

2020

A massively-multiplayer, asynchronous Royal Game of Ur that anyone with a GitHub account can play. The board is the README of the repository. Entirely automated with GitHub actions. Players make their moves by clicking links on the README, which creates an Issue in the repository, which triggers a GitHub action to make the move and update the board state. You can play!
